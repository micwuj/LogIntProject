Emulator

To run virtual device
  1. Make sure that your machine support virtualization: kvm-ok
  2. Switch to root in terminal: sudo -s
  3. Run emulator docker server: docker run -d -p 6080:6080 -e EMULATOR_DEVICE="Samsung Galaxy S10" -e WEB_VNC=true --device /dev/kvm --name android-container budtmo/docker-android:emulator_11.0
  4. Go to emulator
